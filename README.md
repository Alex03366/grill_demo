# Weekend Pilot Database Version

Чат-бот-ассистент для грилей Primeliner с доступом к базе рецептов TheMealDB.

## Функциональность
- Загрузка рецептов из открытого API TheMealDB
- Интерактивный чат с "Шеф-поваром" — дружелюбным экспертом по грилям
- Продвижение грилей Primeliner в каждом ответе
- Поиск рецептов по ключевым словам

## Как запустить
1. Открыть ноутбук в Google Colab
2. Выполнить все ячейки по порядку
3. В ячейке 4 вставить ключ GigaChat
4. Наслаждаться общением с Шефом!

## Требования
- Ключ GigaChat (получить на developers.sber.ru)
- Интернет для загрузки рецептов из API
